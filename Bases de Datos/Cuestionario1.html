Questionario 1:

BD vs DBMS

Preguntas

¿Qué es entonces una base de datos?
De acuerdo con Dámian Pérez en el 2007 una base de datos es un “almacén” que nos permite guardar grandes cantidades de información de forma organizada para que luego podamos encontrar y utilizar fácilmente.
¿En qué casos es conveniente usar bases de datos?
Cuando se tiene una gran y creciente necesidad de almacenar datos, pues aparte de ser almacenados serán organizados y de fácil acceso.
¿Qué es un sistema de gestión de base de datos - DBMS?
Un sistema de gestión de datos o DBMses un software desarrollado para ser la interfaz entre la base de datos, el usuario y las aplicaciones de la computadora
Enlista y explica las funciones/responsabilidades que tienen los DBMS.
  • Control sobre redundancia
  • Consistensia de datos
  • Compartir Datos
  • Mantenimiento de Estándares
  • Mejora en la integridad de datos
  • Mejorar la seguridad de los datos
  • Mejorar la accesibilidad de datos
  • MEjorar la productividad
  • Mejorar el mantenimiento de los datos
  • Mejorar el servicio de copia de seguridad
Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
hace 3 años junto a unos compañeros de prepratoria fundamos la sede de una organización nacional en Morelia, y tuvimos que administrar muchos eventos, participantes, miembros, etc. Lo cual fue muy complicado en su tiempo; pero con estos conocimientos hubiera sido cuestión de hacer una base de datos cumpliendo todo lo antes especificado.
