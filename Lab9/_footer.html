Preguntas


¿Qué hace la función phpinfo()?
Muestra gran cantidad de información sobre el estado actual de PHP. Incluye información sobre las opciones de compilación y extensiones de PHP, versión de PHP, información del servidor y entorno (si se compiló como módulo), entre otras funciones.
¿Qué cambios tendrías que hacer en la configuración del servidor para que pudiera ser apto en un ambiente de producción?
Instalar una aplicación de entorno de servidor como XAMPP, que incluya Apache, MySQL y PHP para poder usar y manejar el servidor.
¿Cómo es que si el código está en un archivo con código html que se despliega del lado del cliente, se ejecuta del lado del servidor? Explica.
Los archivos HTML y PHP se ejecutan desde el servidor, una vez ejecutados y procesados adecuadamente se le envia una respuesta al usuario con el HTML que se despliega en su navegador.
